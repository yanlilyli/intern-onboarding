## 二、业务全景 · 你需要了解的事

### 2.1 光子工作室群

| 维度 | 说明 |
|------|------|
| 所属 | 腾讯互娱（IEG）旗下游戏工作室群 |
| 代表作 | 和平精英、PUBG Mobile 等 |

| 工作室 | S工作室、天玑工作室等多个项目组 |

### 2.2 S工作室

- 光子旗下核心工作室之一
- 覆盖方向：3A渲染/UE引擎/策划/电竞/美术等
- 当前人员规模约1100人（含校招/社招/实习等）
- 持续大规模招聘中，校招+社招双线并行

### 2.3 开局一课

| 维度 | 说明 |
|------|------|
| 性质 | 面向高校学生的校招品牌活动/课程项目 |
| 周期 | 每年一届，2026届报名期4月1日-30日 |
| 目标 | 2026届目标6000人报名 |
| 核心价值 | 品牌认知建设 + 校招人才池储备 + 转化运营 |
| 合作院校 | 北影、浙传等重点院校 |


## 三、你的工作内容

### 3.1 核心工作模块

| 模块 | 具体内容 | 优先级 |
|------|---------|--------|
| **Sourcing简历** | 各渠道开源搜索，筛选匹配候选人 | 🔴 高 |
| **开局一课转化运营** | 已报名学生的跟进、激活、转化为面试候选人 | 🔴 高 |
| **开局一课品牌前置运营** | 为2027届做提前布局，品牌认知渗透 | 🟡 中 |
| **线下活动策划** | 校园宣讲、行业活动、开放日等 | 🟡 中 |
| **了解业务** | 深入理解游戏行业和各岗位 | ✅ 持续 |

### 3.2 Sourcing工作指南

#### 常用sourcing渠道
- **内部渠道**：Offer系统、简历库、活水推荐
- **外部渠道**：Boss直聘、猎聘、脉脉、LinkedIn、ArtStation（美术类）
- **校园渠道**：应届生论坛、校园就业网、学生社群
- **定向Mapping**：行业通讯录、项目组对标、竞品人才梳理

#### 搜索关键词参考（美术方向为例）
- 3A美术/次世代建模/角色原画/场景概念
- UE5/虚幻引擎/技术美术/TA
- 美术总监/美术主管/Lead Artist

### 3.3 开局一课运营

#### 转化运营（当届）
- 报名数据追踪与分析
- 学员激活（推送/通知/社群运营）
- 课程参与度跟进
- 优秀学员标记 → 推荐面试

#### 品牌前置运营（下一届）
- 目标院校mapping（美术/传媒/计算机）
- KOL/学生干部触达
- 内容种草（公众号/小红书/B站）
- 校园大使计划策划

### 3.4 线下活动策划

- 校园宣讲会（北影/浙传/广美等）
- 行业开放日
- 游戏制作体验营
- 联合院校的workshop

## 八、注意事项

### ⚠️ 保密要求
- 候选人信息严格保密，不得外泄
- 薪资、编制、Offer数据属于高度机密
- 内部决策和团队讨论不对外分享
- 使用公司设备处理工作数据

### 💡 工作习惯建议
- 每日更新sourcing进展到共享文档
- 主动汇报，遇到问题及时沟通
- 做好面试笔记和候选人标签
- 关注行业动态，积累人才认知

## 九、常见问题FAQ

| 问题 | 答案 |
|------|------|
| 不懂游戏行业怎么办？ | 正常！前两周专注学习，多看JD和项目资料 |
| Sourcing找不到合适的人？ | 换关键词、换渠道、问leader拿参考画像 |
| 候选人不回复怎么办？ | 换话术、换时间段、尝试不同渠道触达 |
| 面试流程不清楚？ | 参考SOP文档，或直接问同事 |
